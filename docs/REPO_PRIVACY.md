# How to make the repository private (step-by-step)

Since you asked for a low-friction, GUI-based guide, here are explicit steps to make your GitHub repository private:

1. Open your browser and go to: https://github.com/Lazulis-9/Silence
2. Sign in with your GitHub account if needed.
3. On the repository page, click the "Settings" tab (near the top-right, next to "Insights").
4. Scroll down to the "Danger Zone" section at the bottom of the Settings page.
5. Click "Change repository visibility".
6. Select "Make private", then follow the prompts to confirm (GitHub may ask you to type the repository name to confirm).

Note: Making the repo private will stop public access to the code. If you prefer, I can keep helping guide you through the rest of the recovery steps.
