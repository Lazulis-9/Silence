# ADMIN_MANUAL.md

管理员（Madam Lazulis）工作流：

1. 审批新用户
- 登录 Dashboard → 审批中心 → 查看问卷 → 审核并选择通过或拒绝
- 通过时可填写备注并触发激活邮件

2. 审批充值
- 充值中心查看待处理请求 → 验证付款凭证 → 审批到账（系统增加 LC）

3. 审批商品上架
- 商品审核列表 → 查看商品详情与卖家信息 → 通过/拒绝

4. 管理关系图
- 查看任意用户的关系图（仅管理员有解密权限）

5. 系统设置
- 修改货币汇率、页面文案、节点配置

6. 管理密钥
- 管理员密钥用于限制代码层变更：保管好私钥，不要在公开地方保存
