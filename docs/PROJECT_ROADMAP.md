# PROJECT_ROADMAP.md

短期（2 周） - 核心交付：
1. 问卷与注册审批流
2. 邮箱绑定与验证
3. 管理员 Dashboard（审批界面）
4. LC 货币：充值申请与管理员到账确认
5. 基本社区（发帖、交互、认同、记录）
6. 商品上架申请与购买流程
7. 私有关系导图（基础可视化）
8. 部署脚本与说明

中期（1-3 月）：
1. 私信增强（加密/端对端可选）
2. 关系导图高级布局与导出
3. 性能优化（缓存、CDN、读写分离）
4. 多节点支持及同步策略

长期（3-12 月）：
1. 移动端 App（React Native / Expo）
2. 更复杂的分析仪表盘
3. 自动化审批规则（仍保留人工最终权限）
